# 📈 한국 주식 위젯

실시간 한국 주식 현재가를 화면에 항상 띄워두는 미니 위젯 프로그램입니다.

<p align="center">
  <img src="docs/screenshot-widgets.png" alt="여러 종목 위젯이 세로로 정렬된 모습. 장 외 시간에는 sparkline 자리에 최근 30일 일봉 캔들 차트가 표시된다." width="320">
</p>

> 기본값으로 화면 우상단에 종목별 위젯이 세로 정렬되며, 드래그로 모니터 어디에든 자유롭게 배치할 수 있습니다. 장중에는 당일 분봉 sparkline 이, 장 외 시간·주말·공휴일에는 자동으로 최근 30일 일봉 캔들 차트로 폴백됩니다.

---

## ⚙️ 설치 방법

### 1. 라이브러리 설치
```bash
pip install PyQt6 requests
```
또는
```bash
pip install -r requirements.txt
```

### 2. 실행
```bash
python stock_widget.py
```

---

## 🖱️ 사용법

### 위젯
| 동작 | 설명 |
|------|------|
| **좌클릭** | 평단가·수량·투자원금·평가금액·평가손익·수익률 확장 (5초 후 자동 축소) |
| 드래그 | 위젯을 원하는 위치로 이동 (이동 시 클릭으로 인식되지 않음) |
| **우클릭** | 수정 / 삭제 메뉴 |

- 위젯 가로폭은 가장 긴 종목명 기준으로 모든 위젯에 통일 적용됩니다.
- 현재가 옆에 등락률(▲/▼)이 함께 표시되며, 상승 시 빨강 · 하락 시 파랑으로 강조됩니다.

### 시스템 트레이 (작업표시줄 우측)

**좌클릭** → 모든 위젯 표시 / 숨김 빠른 토글

**우클릭 메뉴**
| 메뉴 | 설명 |
|------|------|
| ➕ **종목 추가** | 새 종목을 한 건씩 추가 |
| 📋 **종목 관리** | 보유 종목을 표 형태로 일괄 관리 (추가·수정·삭제·드래그 정렬) |
| 🙈 **숨기기** / 👀 **표시하기** | 모든 위젯 표시/숨김 토글 |
| 📐 **위치 초기화** | 각 위젯이 현재 위치한 모니터의 **우상단**에 세로 정렬 |
| ❌ **종료** | 프로그램 종료 |

### 📋 종목 관리 창

여러 종목을 한 화면에서 일괄 편집할 수 있는 다이얼로그입니다.

| 동작 | 설명 |
|------|------|
| **➕ 추가** | 새 종목 추가 (코드로 종목명 자동 조회, 중복 거부) |
| **✏ 수정** / 더블클릭 | 선택한 행의 평단가·수량 편집 |
| **🗑 삭제** | 선택한 행 삭제 (확인 절차 포함) |
| **행 드래그** | 종목 순서 변경 → 다음 "위치 초기화" 시 새 순서대로 정렬 |
| **확인** | 변경 사항을 위젯과 `stocks.json` 에 반영 |
| **취소** | 모든 변경 사항 폐기 (원본 보존) |

---

## 🖥️ 멀티 모니터 지원

- 위젯은 어느 모니터에서나 자유롭게 이동·배치할 수 있습니다.
- **위치 초기화**는 각 위젯이 *현재 속한 모니터*의 우상단에 정렬하므로, 멀티 모니터 환경에서 위젯이 엉뚱한 화면으로 이동하지 않습니다.
- 작업 표시줄 영역을 피하기 위해 `availableGeometry` 기준으로 좌표를 계산합니다.

## 📁 파일 구조
```
stock_widget.py   ← 메인 프로그램
requirements.txt  ← 필요 라이브러리 목록
stocks.json       ← 종목 설정 자동 저장 (실행 후 생성)
```

`stocks.json` 에는 종목별 코드·종목명·평단가·수량·화면 좌표가 저장되며, 종료 시 위치가 자동 저장됩니다.

## 🖥️ .exe 파일로 변환하기 (배포용)

Python 없이 실행할 수 있는 단독 실행 파일을 만들려면:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name 주식위젯 stock_widget.py
```

생성된 `dist/주식위젯.exe` 파일을 사용하면 됩니다.
